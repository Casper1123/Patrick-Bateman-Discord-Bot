# Patrick Bateman - Discord Bot v3
This folder (not branche, because I forgot to make a branche) is used for the rewrite of the Discord Bot hosted in this project.
In this particular case, even though it is not entirely visible in the git history, this is the third rewrite since starting.
This rewrite uses actual education knowledge as opposed to still learning how to program. The application of these ideas might not be perfect.

## Folder structure

### configuration
Contains files related to active configuration. As of time of writing, this part of the project is still to be made.

### data
Contains template interfaces as well as database implementations required by other program components.

### discorduser
Contains the discord bot user as well as the mounted command cogs as well as the logger. As of time of writing, the logger hasn't been written yet, but will be used for moderation purposes.

### utilities
Random helper utilities, like easy file read and writing, or Exceptions and turning messages into embeds.

### piss
Procedural Instruction Sequence String (PISS); this folder contains the compiler, executor as well as testing tools for related strings.
Still requires heavy development. Might be turned into individual repository to allow for usage in other projects, at which point this will become a submodule that will be modified.
